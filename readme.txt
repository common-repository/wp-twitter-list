Show last Twitt news in list with fade effect.

== Description ==
Show last Twitt news in list with fade effect.

This plugin create a listing of tweet inside tag `ul`, and you can see it inline.

By option page, you can define the duration of fade effect, the number of tweet to see and other things.


= Basic Usage =

If you wanto see the tweet inside your post/page can write `[wp_tweet_list]`.

If you want call this plugin inside you template page .php, can write `<?php echo do_shortcode("[wp_tweet_list]"); ?>`


= Features =

* there is shortcode
* is very customizable

== Installation ==

1. First copy the plugin directory inside wp-contents/plugins directory of your wordpress site.
2. Activate the plugin through the 'Plugins' menu in WordPress.
